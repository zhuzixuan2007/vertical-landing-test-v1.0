# 快速开始指南

## 5分钟上手新的保存系统

### 第一次启动

1. **编译游戏**
   ```cmd
   build.cmd
   ```

2. **运行游戏**
   ```cmd
   vertical-landing-test.exe
   ```

3. **你会看到主菜单**
   ```
   ┌─────────────────────────────┐
   │   VERTICAL LANDING          │
   │      SIMULATOR              │
   ├─────────────────────────────┤
   │                             │
   │    ▶ NEW GAME               │
   │      EXIT                   │
   │                             │
   └─────────────────────────────┘
   ```

4. **选择 NEW GAME**
   - 使用 ↑↓ 方向键选择
   - 按 Enter 确认

### 构建你的火箭

5. **火箭构建界面**
   - 默认已有一个基础火箭（引擎+燃料箱+鼻锥）
   - 可以添加更多零件（可选）
   - 按 Space 发射

### 飞行和自动保存

6. **发射火箭**
   - 火箭会自动起飞
   - 左下角会显示各种数据

7. **观察自动保存**
   - 等待10秒
   - 左下角会出现 "AUTOSAVED" 提示
   - 这表示游戏已自动保存

### 测试继续游戏

8. **退出游戏**
   - 按 ESC 键退出

9. **重新启动**
   - 再次运行 `vertical-landing-test.exe`

10. **你会看到新的菜单**
    ```
    ┌─────────────────────────────┐
    │   VERTICAL LANDING          │
    │      SIMULATOR              │
    ├─────────────────────────────┤
    │                             │
    │    ▶ CONTINUE GAME          │
    │      TIME: 00:05 | PARTS: 3 │
    │                             │
    │      NEW GAME               │
    │      WARNING: WILL DELETE   │
    │                             │
    │      EXIT                   │
    │                             │
    └─────────────────────────────┘
    ```

11. **选择 CONTINUE GAME**
    - 游戏会恢复到你上次的状态
    - 火箭位置、速度、燃料都会保持不变

## 常用操作

### 菜单导航
- **↑↓**: 选择选项
- **Enter/Space**: 确认
- **ESC**: 退出

### 游戏中
- **Tab**: 切换自动/手动模式
- **Space**: 发射（发射前）
- **WASD**: 姿态控制（手动模式）
- **Shift/Ctrl**: 油门控制
- **1-4**: 时间加速
- **H**: 显示/隐藏HUD
- **ESC**: 退出游戏

## 保存系统特性

### 自动保存
- ✅ 每10秒自动保存
- ✅ 左下角显示提示
- ✅ 不影响游戏流畅度
- ✅ 仅在发射后保存

### 继续游戏
- ✅ 一键恢复上次状态
- ✅ 显示存档信息
- ✅ 跳过构建界面
- ✅ 完整恢复所有数据

### 新游戏
- ✅ 开始全新游戏
- ✅ 自动删除旧存档
- ✅ 显示警告提示
- ✅ 进入构建界面

## 存档文件

### 位置
```
游戏目录/rocket_save.dat
```

### 备份建议
定期复制 `rocket_save.dat` 到安全位置：
```cmd
copy rocket_save.dat rocket_save_backup.dat
```

### 恢复备份
```cmd
copy rocket_save_backup.dat rocket_save.dat
```

## 故障排除

### 问题：看不到主菜单
**解决方案**:
1. 确认游戏正确编译
2. 检查OpenGL驱动是否更新
3. 查看控制台错误信息

### 问题：无法保存
**解决方案**:
1. 检查游戏目录写入权限
2. 确认磁盘空间充足
3. 以管理员身份运行

### 问题：加载失败
**解决方案**:
1. 删除 `rocket_save.dat`
2. 重新开始新游戏
3. 如果问题持续，查看详细文档

# 保存系统测试指南

## 测试步骤

### 1. 首次启动测试
**预期行为**: 
- 显示主菜单
- 只显示"NEW GAME"和"EXIT"选项（没有"CONTINUE GAME"）
- 星空背景动画正常

**测试步骤**:
```
1. 编译并运行程序
2. 观察主菜单界面
3. 使用上下键导航菜单
4. 选择"NEW GAME"进入游戏
```

### 2. 火箭构建测试
**预期行为**:
- 显示默认火箭配置（引擎+燃料箱+鼻锥）
- 可以添加/删除零件
- 统计数据正确显示

**测试步骤**:
```
1. 在构建界面浏览零件
2. 添加几个零件
3. 按Space发射
```

### 3. 自动保存测试
**预期行为**:
- 发射后每10秒自动保存
- 左下角显示"AUTOSAVED"提示（持续3秒）
- 保存文件 rocket_save.dat 被创建

**测试步骤**:
```
1. 发射火箭
2. 等待10秒
3. 观察左下角是否出现"AUTOSAVED"提示
4. 检查游戏目录是否生成 rocket_save.dat 文件
```

### 4. 继续游戏测试
**预期行为**:
- 关闭游戏后重新启动
- 主菜单显示"CONTINUE GAME"选项
- 显示存档信息（时间、零件数）
- 加载后恢复到上次状态

**测试步骤**:
```
1. 在游戏中飞行一段时间（至少30秒）
2. 按ESC退出游戏
3. 重新启动程序
4. 观察主菜单是否显示"CONTINUE GAME"
5. 选择"CONTINUE GAME"
6. 验证火箭位置、速度、燃料等是否正确恢复
```

### 5. 新游戏覆盖测试
**预期行为**:
- 有存档时选择"NEW GAME"会显示警告
- 确认后删除旧存档
- 进入火箭构建界面

**测试步骤**:
```
1. 在有存档的情况下启动游戏
2. 选择"NEW GAME"
3. 观察是否显示"WARNING: WILL DELETE SAVE"
4. 按Enter确认
5. 验证进入构建界面
6. 检查旧存档是否被删除
```

### 6. 游戏结束保存测试
**预期行为**:
- 火箭着陆或坠毁时自动保存
- 控制台显示"GAME SAVED"

**测试步骤**:
```
1. 让火箭着陆或坠毁
2. 观察控制台输出
3. 验证存档文件被更新
```

### 7. 存档数据完整性测试
**预期行为**:
- 所有关键数据正确保存和加载
- 火箭装配、位置、速度、燃料等完全一致

**测试步骤**:
```
1. 记录当前游戏状态：
   - 火箭零件配置
   - 位置坐标
   - 速度
   - 燃料量
   - 高度
   - 控制模式（自动/手动）
2. 退出并重新加载
3. 对比所有数据是否一致
```

### 8. 边界情况测试

#### 8.1 发射前不保存
**测试**: 在PRE_LAUNCH状态等待，不应该自动保存

#### 8.2 快速退出
**测试**: 发射后立即退出（10秒内），验证是否有存档

#### 8.3 长时间游戏
**测试**: 游戏运行超过5分钟，验证多次自动保存是否正常

#### 8.4 存档文件损坏
**测试**: 
```
1. 手动编辑 rocket_save.dat，破坏格式
2. 重新启动游戏
3. 尝试加载存档
4. 验证是否有错误处理
```

## 常见问题排查

### 问题1: 主菜单不显示
**可能原因**:
- OpenGL初始化失败
- 渲染器未正确创建
- 字体渲染问题

**排查步骤**:
1. 检查控制台错误信息
2. 验证OpenGL版本支持
3. 测试基本渲染功能

### 问题2: 无法保存
**可能原因**:
- 文件写入权限不足
- 磁盘空间不足
- 路径错误

**排查步骤**:
1. 检查游戏目录权限
2. 手动创建测试文件验证写入权限
3. 查看控制台错误信息

### 问题3: 加载失败
**可能原因**:
- 存档文件损坏
- 版本不兼容
- 数据格式错误

**排查步骤**:
1. 用文本编辑器打开 rocket_save.dat 检查格式
2. 删除存档重新测试
3. 检查版本号是否匹配

### 问题4: 数据不一致
**可能原因**:
- 保存/加载逻辑错误
- 数据类型转换问题
- 遗漏某些字段

**排查步骤**:
1. 对比保存和加载的代码
2. 添加调试输出验证每个字段
3. 检查数据类型是否匹配

## 性能基准

### 保存操作
- 预期时间: < 10ms
- 文件大小: < 5KB
- CPU占用: 可忽略

### 加载操作
- 预期时间: < 50ms
- 内存占用: < 1MB
- 启动延迟: < 100ms

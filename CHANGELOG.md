# 更新日志

## [1.1.0] - 2026-03-02

### 新增功能 ✨

#### 游戏保存系统
- **启动菜单**: 添加了美观的主菜单界面
  - 星空背景动画
  - 菜单选项导航（继续游戏/新游戏/退出）
  - 存档信息显示（游戏时间、零件数量）
  - 选中项高亮和脉冲动画

- **自动保存**: 游戏运行时自动保存
  - 每10秒自动保存一次
  - 左下角显示"AUTOSAVED"提示（持续3秒）
  - 仅在发射后开始保存
  - 不影响游戏性能

- **继续游戏**: 一键恢复上次游戏状态
  - 完整恢复火箭装配
  - 恢复位置、速度、姿态
  - 恢复燃料和控制状态
  - 恢复游戏时间和任务状态
  - 跳过火箭构建界面

- **新游戏**: 开始全新游戏
  - 自动删除旧存档
  - 显示警告提示
  - 进入火箭构建界面

- **游戏结束保存**: 着陆或坠毁时自动保存最终状态

### 技术改进 🔧

#### 新增文件
- `src/save_system.h` - 保存/加载核心功能
- `src/menu_system.h` - 启动菜单系统

#### 修改文件
- `src/project.cpp` - 集成保存系统和菜单系统
  - 添加主菜单循环
  - 添加自动保存逻辑
  - 添加存档加载逻辑
  - 添加保存指示器

#### 数据格式
- 使用文本格式存储，便于调试
- 包含版本号，便于未来兼容性
- 完整保存游戏状态

### 用户体验改进 🎨

- **无缝体验**: 自动保存，无需手动操作
- **快速恢复**: 一键继续上次游戏
- **清晰反馈**: 保存提示和存档信息

## 计算公式验证

### 质心公式
```
CoM = Σ(mi * pi) / Σ(mi)
```
其中:
- mi = 部件质量 (dry_mass + fuel_capacity)
- pi = 部件中心位置 (stack_y + height/2)

✅ **验证通过** - 所有测试用例的手动计算与函数输出一致

### 升力中心公式
```
CoL = Σ(Ai * pi) / Σ(Ai)
```
其中:
- Ai = 气动部件的有效面积
- pi = 部件中心位置 (stack_y + height/2)

✅ **验证通过** - 面积加权计算正确，鼻锥和尾翼识别准确

### 推力中心公式
```
CoT = Σ(Ti * pi) / Σ(Ti)
```
其中:
- Ti = 引擎推力
- pi = 引擎中心位置 (stack_y + height/2)

✅ **验证通过** - 推力加权计算正确，引擎和助推器识别准确

## 边界情况处理

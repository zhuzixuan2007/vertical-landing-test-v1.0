# 实现计划：火箭中心点可视化

## 概述

本实现计划将火箭建造车间的中心点可视化功能分解为增量式的编码任务。实现将按照以下顺序进行：首先建立数据结构和计算逻辑，然后添加可视化渲染，最后集成UI控制。每个任务都会引用相应的需求条款，确保完整的需求覆盖。

  - 将 CenterVisualizationState 成员添加到 BuilderState 结构体中
  - 初始化所有状态字段为默认值（所有显示标志为false）
  - _需求: 1.5, 2.5, 3.5_

    - 实现 calculateCenterOfMass 函数
    - 遍历所有部件，累加质量加权位置：CoM = Σ(mi * pi) / Σ(mi)

  - [ ]* 2.3 为质心计算编写属性测试

    - 实现 calculateCenterOfLift 函数
    - 识别气动部件（鼻锥、尾翼）
    - 计算面积加权位置：CoL = Σ(Ai * pi) / Σ(Ai)
    - 处理无气动表面的情况
    - _需求: 6.2_

  - [ ]* 2.5 为升力中心计算编写属性测试

  - [x] 2.6 实现推力中心计算函数

    - 识别引擎部件（引擎、助推器）
    - 计算推力加权位置：CoT = Σ(Ti * pi) / Σ(Ti)
    - 处理无引擎的情况
    - _需求: 6.3_
